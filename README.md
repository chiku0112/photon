# Photon Classroom App

This is a classroom app which has A Teacher and A Student Usertype. Main actions that can be performed are : 
- Creating classroom for respective subjects
- Adding students in a Classroom
- Creating an Assignment 
- Booking a Counselling Session

#### Setup

You will first have to create a `config/keys.js` file wherin you have to add `mongoUri` for connecting to db server. Then follow following steps to setup the project

- `npm run install`
- `cd client && npm install`
- `npm run dev` this will start the backend server(run this from the root of the directory)
- `cd client && npm run start` this will start the react frontend.
